# Perishable Goods Management: Exploratory Data Analysis

## 📌 Project Overview
This project analyzes 100,000 transactional records of perishable goods to identify factors influencing spoilage risk, waste cost, and profitability.

## 🎯 Objectives
- Analyze revenue and profit patterns
- Identify key drivers of spoilage
- Understand impact of waste cost on profit
- Provide data-driven business recommendations

## 📊 Key Insights
- Pharmaceuticals generate the highest revenue.
- Ready-to-Eat products have the highest spoilage rate.
- Temperature deviation moderately increases spoilage risk.
- Waste cost strongly reduces profitability (-0.65 correlation).
- Shelf life alone does not significantly influence spoilage risk.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📁 Files Included
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
- Project Report (.pdf)

## 📈 Business Impact
This analysis demonstrates that operational factors such as temperature control and waste management significantly impact profitability in perishable goods management.
